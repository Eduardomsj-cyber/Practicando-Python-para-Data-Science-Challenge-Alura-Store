# Practicando Python para Data Science — Challenge Alura Store

Análisis de ventas y rendimiento de tiendas para apoyar una decisión de negocio.

## Descripción
En este proyecto resuelvo el desafío **Alura Store**, cuyo objetivo es ayudar al Sr. Juan a decidir **qué tienda vender** para invertir en un nuevo negocio.  
Para ello, analizo métricas de ventas y desempeño (ingresos, volumen, categorías, calificaciones, etc.) y presento conclusiones basadas en datos.

## Objetivos
- Comparar el rendimiento de varias tiendas con métricas clave.
- Identificar la tienda con menor desempeño relativo.
- Entregar hallazgos y recomendación final sustentada en el análisis.

## Tecnologías
- Python
- Pandas (si aplica)
- Matplotlib/Seaborn (si aplica)
- Jupyter Notebook (si aplica)

## Estructura del repositorio

- `data/` — dataset(s) (si aplica)

- `reports/` — resultados, conclusiones o exportables (si aplica)
